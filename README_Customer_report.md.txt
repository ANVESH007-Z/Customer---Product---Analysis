# Customer Report SQL Project

## Overview

This project contains a SQL-based Customer Analytics Report designed to analyze customer behaviour, purchasing patterns, and overall customer performance.

The report transforms raw sales and customer data into meaningful business insights that help organizations understand:

* Customer purchasing behaviour
* Revenue contribution by customers
* Customer engagement trends
* Customer segmentation
* Retention and recency analysis

---

# Project Objectives

The main objective of this project is to build an end-to-end SQL analytical report that:

* Analyzes customer purchasing activity
* Measures customer value and contribution
* Identifies high-value and inactive customers
* Generates customer-focused KPIs
* Supports customer-driven business decisions

# Query Structure

## 1. Base Query

Retrieves and joins:

* Customer data
* Sales transactions

Creates the foundational dataset for analysis.

## 2. Customer Aggregation

Calculates:

* Total customer sales
* Total orders
* Quantity purchased
* Purchase frequency
* Customer lifespan

## 3. Final Report

Builds the final customer analytics report by:

* Calculating KPIs
* Measuring customer recency
* Calculating average revenue metrics
* Segmenting customers based on spending behaviour
